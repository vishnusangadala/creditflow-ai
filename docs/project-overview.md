Problem:

Helping financial analysts to take decisions faster on Loan approval and do risk analysis better.

Goal:

(Automating reading and extracting useful info for metric calculation and decision making in financial documents (for ex: 100's of pages))

Workflow:

Upload files -> information extraction (extraction agent) -> Risk Analysis (Risk Agent) -> credit memo agent -> verifier agent -> human in the loop

(Auditing) - Agentic workflow audits the complete analysis and reasons about the actual work it has been done.
(verification) - Sometimes LLMs hallucinate and comeup with some wrong values or something like that, so verification agent has its role to verfiy and check if extracted valeus are legit or not.
